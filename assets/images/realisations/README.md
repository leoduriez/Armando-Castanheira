## Format et optimisation

- **Format** : Toutes les images doivent être en `.webp`
- **Dimensions recommandées** :
  - Images simples : 1200x900px (ratio 4:3)
  - Images doubles : 800x800px (ratio 1:1)
  - Header : 1920x1080px minimum
- **Compression** : Optimisées pour le web (qualité 80-85%)
- **Poids** : Maximum 200-300 KB par image

## Conversion en WebP

Pour convertir vos images en WebP, vous pouvez utiliser :

### En ligne de commande (avec cwebp) :
```bash
cwebp -q 85 input.jpg -o output.webp
```

### Avec un outil en ligne :
- https://squoosh.app/
- https://cloudconvert.com/

### Avec Photoshop :
- Installer le plugin WebP
- Exporter en WebP avec qualité 80-85%
